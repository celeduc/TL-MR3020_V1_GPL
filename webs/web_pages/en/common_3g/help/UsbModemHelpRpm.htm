3G/4G USB Modem Settings Help

You can upload the configure file of your USB modem on this page.

Note: There are already much 3G/4G USB modem information embedded in the Device. The USB modem parameters will be set automatically if the card is supported by the Device. But when the Device finds the card you just insert "unknown" to it, it will prompt you to upload 3G/4G USB modem configuration file. The Device can identify your "unknown" card if the correct parameters are shown in the list. We suggest you do the settings only in such circumstance. Do not unplug the 3G/4G modem, for the device is running commands in the list.

To add 3G/4G USB Modem entries:

  1. Download a most recent 3G/4G USB modem configuration file from our website ( ).
  2. Click the Add New... button to upload the configuration.
  3. Select the path name where you save the downloaded file on the computer into the File blank.

To delete a 3G/4G USB Modem entry:

  1. Find the desired entry in the table.
  2. Click Delete as desired on the Delete column.

Click the Delete All button to delete all the entries in the table.
